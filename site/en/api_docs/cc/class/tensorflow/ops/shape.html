tensorflow::ops::Shape

#include <array_ops.h>

Returns the shape of a tensor.

Summary

This operation returns a 1-D integer tensor representing the shape of input.

For example:

# 't' is [[[1, 1, 1], [2, 2, 2]], [[3, 3, 3], [4, 4, 4]]]
shape(t) ==> [2, 2, 3]

Arguments:

  • scope: A Scope object

Returns:

  • Output: The output tensor.
