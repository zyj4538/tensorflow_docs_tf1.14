tensorflow::ops::Zeta

#include <math_ops.h>

Compute the Hurwitz zeta function \((x, q)\).

Summary

The Hurwitz zeta function is defined as:

\((x, q) = {n=0}^{} (q + n)^{-x}\)

Arguments:

  • scope: A Scope object

Returns:

  • Output: The z tensor.
