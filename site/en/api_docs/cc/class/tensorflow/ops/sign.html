tensorflow::ops::Sign

#include <math_ops.h>

Returns an element-wise indication of the sign of a number.

Summary

y = sign(x) = -1 if x < 0; 0 if x == 0; 1 if x > 0.

For complex numbers, y = sign(x) = x / |x| if x != 0, otherwise y = 0.

Arguments:

  • scope: A Scope object

Returns:

  • Output: The y tensor.
