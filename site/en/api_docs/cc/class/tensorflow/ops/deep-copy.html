tensorflow::ops::DeepCopy

#include <array_ops.h>

Makes a copy of x.

Summary

Arguments:

  • scope: A Scope object
  • x: The source tensor of type T.

Returns:

  • Output: y: A Tensor of type T. A copy of x. Guaranteed that y is not an alias of x.
