tensorflow::ops::ShardedFilename

#include <io_ops.h>

Generate a sharded filename.

Summary

The filename is printf formatted as

s-%05d-of-%05d, basename, shard, num_shards.

Arguments:

  • scope: A Scope object

Returns:

  • Output: The filename tensor.
