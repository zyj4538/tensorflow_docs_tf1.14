tensorflow::ops::DataFormatVecPermute

#include <nn_ops.h>

Returns the permuted vector/tensor in the destination data format given the.

Summary

  • scope: A Scope object
  • x: Vector of size 4 or Tensor of shape (4, 2) in source data format.

Returns:
